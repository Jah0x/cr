# Reporting spec

## Core metrics
- turnover (paid sales total)
- cogs (sum sale items cogs_total)
- gross_profit = turnover - cogs
- margin = gross_profit / turnover

## Slices
- by category
- by brand
- by line
- by cashier (optional)

## Stock analytics
- current on-hand
- low stock alerts by threshold
- sales velocity (optional later)
